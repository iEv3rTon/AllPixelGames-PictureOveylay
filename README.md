# How to setup

1. Install tampermonkey addon for your browser from - <https://www.tampermonkey.net/> .
2. Open <https://github.com/iEv3rTon/AllPixelGames-PictureOveylay/raw/main/AllPixelGames-PictureOveylay-loader.user.js> .
3. Click "Install".
4. Enjoy.
